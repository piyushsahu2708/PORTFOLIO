# **App Name**: Portfolio Pro

## Core Features:

- Profile Showcase: Display a professional headshot, name, roles, and a brief summary to immediately grab recruiters' attention, using the provided image for the profile picture.
- Interactive Experience Display: Present past work experiences with responsibilities, including ITJOBXS internship, and AWS Solutions Architecture highlights with verifiable achievements. Tech stack is explicitly specified, including integrations (e.g. Google reCAPTCHA)
- Skills Overview: Categorize and display technical skills (programming languages, frameworks, databases, tools) for easy evaluation by recruiters.
- Project Demonstrations: Showcase SkillMirror, PractorVision, and MediTract Pro projects, detailing the problems they solve, features implemented, tech stacks used, and links to GitHub and live demos.
- Resume & Contact Access: Provide direct access to download the resume, along with contact information (email, phone, LinkedIn, GitHub) for immediate communication.
- SEO Optimization Tool: An AI tool to ensure that the displayed content and meta tags are optimized with recruiter-friendly keywords for improved search visibility. The tool provides recommended keywords based on analysis of competitor portfolios and job descriptions. The tool uses reasoning about what is important to the user to decide if the keyword should be included.

## Style Guidelines:

- Primary color: Deep Indigo (#4B0082) to convey professionalism and intelligence.
- Background color: Very light gray (#F5F5F5), creating a clean and modern backdrop to ensure readability and focus on content.
- Accent color: Teal (#008080), highlighting skills and project links.
- Body and headline font: 'Inter' sans-serif font for a modern, neutral, and highly readable experience.
- Code font: 'Source Code Pro' for displaying code snippets.
- A clean, single-page layout with clear sections for Hero, About Me, Experience, Skills, Projects, Achievements, and Contact, ensuring easy navigation.
- Subtle animations for section transitions and skill highlighting.
